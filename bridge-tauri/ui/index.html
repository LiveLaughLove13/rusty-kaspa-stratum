How to start the Stratum bridge

Short version: open Connection, set your node address and Stratum port, then Save and Start bridge. The dashboard opens in this window. Use Stop bridge when finished (or Bridge → Stop bridge in the menu). Reloading (F5) while the bridge is running reconnects to the dashboard.

Quick start

Typical single-listener setup:

  1. Enter your Kaspa node address under Essential.
  2. Set Stratum listen port (and difficulty if needed) under Mining ports.
  3. Save configuration, then Start bridge at the bottom.
